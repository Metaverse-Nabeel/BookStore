## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

Node.js
Visual Studio Code
Browser

### Setup

git clone https://github.com/Metaverse-Nabeel/Math-Magicians.git

cd Math-Magicians

### Install

npm install

### Usage

npm run start

### Run tests

npm test

### Deployment

npm run build
